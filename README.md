개인적인 설문지 작성 뿐만 아니라, 팀원들과의 공유를 통해 하나의 설문지를 다 함께 작성할 수 있는 **설문지 작성 서비스**입니다. <br />
설문지 응답받기 여부를 설정할 수 있으며, 응답받기가 활성화되어 응답을 받은 설문지는 결과 페이지에서 해당 설문지의 결과를 확인할 수 있습니다. <br />
